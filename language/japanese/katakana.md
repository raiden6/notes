# Katakana

## Basic Katakana
|       | a    | i    | u    | e    | o    |
|-------|------|------|------|------|------|
| **-** | ア (a) | イ (i) | ウ (u) | エ (e) | オ (o) |
| **k** | カ (ka) | キ (ki) | ク (ku) | ケ (ke) | コ (ko) |
| **s** | サ (sa) | シ (shi) | ス (su) | セ (se) | ソ (so) |
| **t** | タ (ta) | チ (chi) | ツ (tsu) | テ (te) | ト (to) |
| **n** | ナ (na) | ニ (ni) | ヌ (nu) | ネ (ne) | ノ (no) |
| **h** | ハ (ha) | ヒ (hi) | フ (fu) | ヘ (he) | ホ (ho) |
| **m** | マ (ma) | ミ (mi) | ム (mu) | メ (me) | モ (mo) |
| **y** | ヤ (ya) |      | ユ (yu) |      | ヨ (yo) |
| **r** | ラ (ra) | リ (ri) | ル (ru) | レ (re) | ロ (ro) |
| **w** | ワ (wa) |      |      |      | ヲ (wo) |
| **n** |      |      | ン (n) |      |      |
